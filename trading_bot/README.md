# 🤖 نظام أبحاث التداول الآلي (Crypto Trading Research)

إطار عمل لاختبار استراتيجيات التداول في العملات الرقمية على منصة OKX،
مصمّم للعمل عبر GitHub Actions. **الهدف الأساسي: البحث والاختبار الصادق
قبل أي مخاطرة بأموال حقيقية.**

## ⚠️ تنبيه مهم

- هذا أداة **بحثية تعليمية**، وليست نصيحة مالية.
- لا توجد استراتيجية تضمن الربح. التداول بالعقود الآجلة قد يؤدي لخسارة كامل رأس مالك.
- **لا تستخدم أموالًا حقيقية** قبل اختبار طويل في وضع المحاكاة (Paper Trading).
- النتائج التاريخية لا تضمن النتائج المستقبلية.

## 📂 البنية

| الملف | الوظيفة |
|-------|---------|
| `config.py` | كل الإعدادات والمعاملات في مكان واحد |
| `data_feed.py` | جلب بيانات الشموع من OKX (بيانات عامة، بلا مفاتيح) |
| `indicators.py` | المؤشرات الفنية: EMA, RSI, ATR |
| `strategy.py` | منطق الإشارات + إدارة المخاطر (حجم الصفقة، الوقف، الهدف) |
| `backtest.py` | محرّك الاختبار التاريخي مع تقرير أداء مفصّل |
| `walkforward.py` | اختبار المتانة عبر شرائح زمنية متعددة (يكشف الحظ من المهارة) |

## 🚀 الاستخدام

```bash
pip install pandas numpy requests

# اختبار تاريخي على فترة واحدة
python backtest.py --candles 3000 --verbose

# اختبار المتانة (الأهم) — عبر فترات متعددة
python walkforward.py --candles 4000 --folds 4
```

## 📊 الاستراتيجية الحالية

استراتيجية اتجاهية متعددة طبقات التأكيد:
1. **الاتجاه:** EMA(50) > EMA(200)
2. **الدخول:** تقاطع EMA(9) فوق EMA(21)
3. **الزخم:** RSI بين 50 و 70
4. **التأكيد:** الحجم أعلى من متوسطه

**إدارة المخاطر:** مخاطرة 2% لكل صفقة، وقف خسارة بـ ATR، نسبة عائد/مخاطرة 1:2،
وقف متحرك.

## 🔬 نتيجة الاختبار الأولية

- اختبار فترة واحدة: نتيجة إيجابية ظاهريًا (+13%).
- **اختبار المتانة (walk-forward): ~0% عبر الفترات** → الاستراتيجية غير متينة
  وتحتاج تطويرًا قبل أي استخدام جاد.

هذا يوضّح أهمية الإطار: **يكشف الحقيقة بدل الأوهام.**
