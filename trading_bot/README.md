## 🏆 الخلاصة العلمية (بعد 16 تجربة كبرى)

اختبرنا **14 عائلة استراتيجية** و**آلاف الصفقات المحاكاة** عبر تحقّق صارم
خارج العيّنة واختبار شرائح زمنية. النتيجة الموحّدة بلا استثناء واحد:

**لا توجد ميزة ربحية مستدامة قابلة للاستخراج من بيانات الأسعار العامة**
(إطار الساعة، بعد الرسوم). كل نتيجة "لامعة" في نافذة واحدة انهارت عند
اختبارها عبر فترات متعددة — انحياز الاختبار المتعدد في أنقى صوره.

| التجربة | النتيجة خارج العيّنة |
|---------|:--:|
| مؤشرات فنية (EMA/RSI/ADX) | ≈ 0% |
| Long + Short | ≈ 0% |
| تحسين آلي (1,215 تركيبة) | −0.1% (Overfitting مثبت) |
| تعلّم آلي (Gradient Boosting، 25 خاصية) | −20.7% (AUC ≈ 0.52) |
| ICT / SMC | −1.9% (تذبذب ±19%) |
| تعلّم معزّز (Q-Learning) | −5.2% |
| النظام الموحّد (10 مكوّنات) | +1.1% (ضجيج) |
| Walk-Forward Optimization | +0.2% (2/7) |
| مختبر 8 عائلات (Donchian/Ichimoku/...) | متوسط −1.8% |
| أفضل إشارة (Momentum Confluence) في اختبار الشرائح | −0.95% (±18%) |
| 🏰 بوت القلعة (أفضل إشارة + كل الحمايات) | **−0.56% لكن ±6.3% فقط** |
| بوتات مرفوعة (تصويت 3 استراتيجيات) | −13.6% |
| رأس مال كامل × رافعة 10x (هدف 1$→10$) | 💥 إفلاس 5/5 |
| مونت كارلو (20,000 مسار، All-in × 10x) | إفلاس 94% |
| 💎 حصاد التمويل (محايد للسعر — بلا تنبؤ) | **+1.6~2.1% سنويًا — الموجب الوحيد** |

**القانونان المستخلصان:**
1. *أنظمة الحماية تدير المخاطرة ولا تصنع ميزة* — بوت القلعة خفّض التذبذب
   3 أضعاف لكنه بقي عند الصفر.
2. *كل ما لا يتنبأ يتفوق على كل ما يتنبأ* — الاستراتيجية الموجبة الوحيدة
   (حصاد التمويل) لا تتنبأ بالسعر إطلاقًا.

## 📂 خريطة الوحدات (29 ملفًا)

### البنية الأساسية
| الملف | الوظيفة |
|---|---|

| الملف | الوظيفة |
|---|---|

## 🚀 التشغيل

```bash
pip install -r requirements.txt

python backtest.py --candles 3000        # اختبار تاريخي
python walkforward.py --folds 4          # اختبار المتانة (الأهم)
python strategy_lab.py                   # مختبر الـ 8 عائلات
python fortress.py                       # بوت القلعة بكل الحمايات
python best_bot.py                       # المقارنة الصادقة النهائية
python montecarlo.py                     # ماذا لو دخلت All-in؟
```

### تفعيل البوت الحيّ (وضع Demo الرسمي من OKX — أموال وهمية)
1. فعّل **Demo Trading** في OKX وأنشئ مفاتيح API تجريبية.
2. أضِف في GitHub: Settings → Secrets → Actions:
   `OKX_API_KEY`, `OKX_SECRET_KEY`, `OKX_PASSPHRASE`
3. الـ workflow يتداول تلقائيًا كل 30 دقيقة بأموال وهمية.

التداول الحقيقي **مقفول عمدًا** خلف `OKX_LIVE=I_ACCEPT_FULL_RISK` —
والأرقام أعلاه تشرح لماذا لا ننصح بفتحه.
