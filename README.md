# Aymanooi.github.io — مصنع أصول رقمية صغيرة (Digital Asset Factory)

نظام لإنتاج أصول رقمية صغيرة (أدوات ويب) **تعمل بالكامل في المتصفح**، عربية أولًا،
مستضافة مجانًا على **GitHub Pages** — بلا خادم وبلا تكلفة تشغيل.

🔗 https://aymanooi.github.io

> **الحالة:** مرحلة التصميم مكتملة. العقود مثبّتة، والأساس المُختبَر جاهز.
> التنفيذ يبدأ بـ"شريحة عمودية" واحدة (صفحة محوّل) قبل أي توسّع.

## الفكرة

بدل بناء أدوات منفصلة، نبني **نظامًا** من ثلاث طبقات (اكتشاف الطلب → تحويله لصفحات →
نشر + SEO)، يعتمد **أنماطًا بارامترية (archetypes)**: قالب واحد + جدول بيانات = عشرات الصفحات.
الهدف: التوسّع إلى 100+ صفحة **دون إعادة هيكلة**.

## البنية الحالية

```
lib/            # دوال نقية معزولة (مُختبَرة) — قلب المنطق
  text-utils.js
  json-utils.js
  image-utils.js
tests/          # اختبارات node:test لكل ملف في lib
assets/
  styles.css    # التنسيق المشترك (ثيم RTL + وضع ليلي)
docs/           # وثائق القرار والتصميم والعقود
package.json
```

## الاختبارات

```bash
node --test
```

كل دالة منطق لها اختبار. المنطق معزول عن الـ DOM ليبقى قابلًا للاختبار والتوسّع.

## المبادئ الحاكمة

- **بلا backend، بلا تكلفة تشغيل** — كل المعالجة في متصفح المستخدم.
- **خصوصية مطلقة** — بيانات المستخدم لا تغادر جهازه.
- **عربية أولًا** (RTL)، أداء فائق، بلا إطار عمل.
- **القيمة قبل الكمّ** — لا تُنشأ صفحة لا تجتاز معايير الفائدة.
