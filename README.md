# Aymanooi.github.io

موقعي الشخصي — يعرض **نظام البحث الكمّي لاستراتيجيات التداول**:
إطار عمل برمجي كامل لاختبار استراتيجيات تداول العملات الرقمية بمنهجية
علمية صارمة.

🌐 **الموقع:** [aymanooi.github.io](https://aymanooi.github.io)

## المحتوى

- **`index.html`** — صفحة العرض الرئيسية للمشروع.
- **`trading_bot/`** — الكود المصدري لنظام البحث الكمّي (Python):
  جلب البيانات، المؤشرات، الاختبار التاريخي، اختبار المتانة، التحسين
  الآلي، والتعلّم الآلي. التفاصيل في
  [`trading_bot/README.md`](trading_bot/README.md).

## الخلاصة العلمية

بعد اختبار أربعة مناهج (من المؤشرات الفنية إلى التعلّم الآلي) بمنهجية
صارمة وتحقّق خارج العيّنة: **لا توجد ميزة ربحية مستدامة** يمكن استخراجها
من بيانات الأسعار العامة بعد خصم الرسوم. قيمة المشروع في النزاهة العلمية
ومنهجية الاختبار، لا في وعود ربح وهمية.

> ⚠️ لأغراض بحثية وتعليمية فقط — ليس نصيحة مالية.
